# Projecte_Estructura_Dades

## Descripció

Projecte académic de l'assignatura d'Estructures de Dades del 2n curs de l'Enginyeria de Dades a la UAB.

## Objectius

- Implementar i comprendre estructures de dades fonamentals
- Aplicar algoritmes eficients de cerca i ordenació
- Analitzar la complexitat temporal i espacial

## Estructura del Projecte

```plaintext
.
├── src/                # Codi font
├── generated_images/   # Imatges i Galeries DB
├── output/             # Resultats de les Galeries
└── README.md
```
